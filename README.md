Local installation
------------------

### Clone repo and install gems

    git clone https://github.com/ministryofjustice/civil-claims-frontend-claimant.git

    cd civil-claims-backend-data-api
    bundle install

### Run tests

    rake seed_user # creates test user in db

    rspec spec
